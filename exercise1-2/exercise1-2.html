Exercise 1.2

Your sketch:

Instructions

Open exercise1-2/sketch.js, add the setup() and draw() functions and create a 600 x 600 canvas.

Write a function called drawTriangle() that has no parameters. Your drawTriangle() function should set fill to a random colour and draw a triangle using mouseX and mouseY as the coordinates for one corner of the triangle. Set the x and y coordinates for the other two corners randomly. We haven't used p5.js's triangle() function much so check the reference (and the previous exercise) if you're not sure how to use it.

Finally, add the appropriate event function to detect mouse clicks and call drawTriangle() whenever the user clicks on the canvas.

You can decide whether all previous triangles should be cleared when the mouse is clicked. If you decide to clear the canvas, don't call background() in draw() for this exercise because it will cover the most recent triangle.

When you are done, check your work using check-exercise1-2.html.

Expected output:

Try implementing the drawTriangle() function first. If you're not sure how to create a random colour, there are examples from recent lectures that should help you.

Implement one of the mouse event functions: mouseClicked(), mousePressed(), or mouseReleased(). It doesn't matter which you choose. In the event function, call drawTriangle().

You may find the following pages of the p5.js reference useful:

  • triangle()
  • mouseClicked()
  • mousePressed()
  • mouseReleased()
